git clone -> clonar o repositório do GitHub para a máquina
git add -> adiciona todas as alterações na branch (ou arquivos específicos)
git commit -m -> cria o commit e dá um nome "x"
git push -> manda as alterações para a main/branch

git pull -> puxa as alterações da main/branch
git status -> mostra tudo que foi alterado (compara main/branch e mostra o que foi alterado)
git checkout branch -> transita entre as branch's 
git checkout -b -> cria uma branch
git reset -> reinicia o processo inteiro (ctrl z)
git merge -> juntar commits, mescla duas branch's
